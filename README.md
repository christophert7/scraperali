# 🛒 AliExpress Product Scraper

أداة Python لاستخراج بيانات المنتجات من أي صفحة داخل **AliExpress** تعرض قائمة منتجات:

- ✅ SuperDeals
- ✅ Bundle Deals
- ✅ Choice
- ✅ Flash Deals
- ✅ نتائج البحث
- ✅ أي صفحة أخرى تعرض قائمة منتجات

تعتمد الأداة على **Playwright** لأنه ينفذ JavaScript ويتعامل مع Lazy Loading / Infinite Scroll، على عكس BeautifulSoup الذي يقرأ HTML الثابت فقط.

---

## 📦 البيانات المستخرجة لكل منتج

| العمود | الوصف |
|---|---|
| Product Name | اسم المنتج |
| Price | السعر الحالي |
| Product URL | رابط المنتج المباشر |
| Image URL | رابط الصورة الرئيسية |
| Store | اسم المتجر (إن وجد) |
| Discount | نسبة الخصم (إن وجدت) |
| Orders | عدد الطلبات / المبيعات (إن وجد) |
| Rating | التقييم (إن وجد) |

النتائج تُحفظ تلقائياً في ثلاثة ملفات: `products.csv` و `products.xlsx` و `products.json` مع **إزالة أي تكرار**.

## ⚙️ طريقة التثبيت

يتطلب Python 3.9 أو أحدث.

```bash
# 1) تثبيت المكتبات المطلوبة
pip install -r requirements.txt

# 2) تنزيل متصفح Chromium الخاص بـ Playwright (مرة واحدة فقط)
playwright install chromium
```

## 🚀 طريقة التشغيل

```bash
python scraper.py
```

سيسألك البرنامج:

```
أدخل رابط الصفحة: https://www.aliexpress.com/w/wholesale-headphones.html
أدخل عدد المنتجات: 100
```

وأثناء التشغيل يعرض التقدم:

```
تم استخراج 20 من 100
```

> ```bash

## 🔢 كيفية تغيير عدد المنتجات

ببساطة أدخل الرقم الذي تريده عند سؤال `أدخل عدد المنتجات:`.
البرنامج يمرر تلقائياً في الصفحة (Infinite Scroll) حتى يجمع العدد المطلوب **ثم يتوقف مباشرة**.
إذا انتهت منتجات الصفحة قبل الوصول للعدد المطلوب، يحفظ ما تم جمعه ويخبرك بذلك.

---

## 🔗 كيفية استخدام روابط SuperDeals و Bundle Deals و Choice

افتح الصفحة المطلوبة في متصفحك، انسخ الرابط من شريط العنوان، والصقه في البرنامج. أمثلة:

| الصفحة | مثال للرابط |
|---|---|
| SuperDeals | `https://www.aliexpress.com/ssr/300000512/nn2024...` أو من قائمة "Super Deals" في الموقع |
| Bundle Deals | `https://www.aliexpress.com/ssr/300000444/BundleDeals...` |
| Choice | `https://www.aliexpress.com/p/choice-main/index.html` |
| نتائج البحث | `https://www.aliexpress.com/w/wholesale-<كلمة البحث>.html` |

> ملاحظة: روابط الحملات تتغير باستمرار، لذلك الأفضل دائماً نسخ الرابط مباشرة من المتصفح. الأداة تتعامل مع أي صفحة تعرض قائمة منتجات بغض النظر عن شكل الرابط.

## 🧠 كيف تتعامل الأداة مع تغيّر تصميم الموقع؟

الأداة **لا تعتمد على Selector واحد**، بل تعمل بطبقتين:

1. قائمة **Selectors احتياطية** لبطاقات المنتجات المعروفة في AliExpress، تُجرَّب بالترتيب.
2. **خطة بديلة عامة**: أي رابط يحتوي على `/item/` يُعتبر منتجاً، ثم تُستخرج البيانات من الحاوية المحيطة به باستخدام أنماط نصية (Regex) للسعر والخصم والطلبات والتقييم.

بهذا تستمر الأداة في العمل حتى لو غيّرت AliExpress أسماء الـ CSS Classes.

كما أنها:
- تتجاهل الإعلانات والعناصر غير الخاصة بالمنتجات.
- تتجاوز أي منتج يحدث فيه خطأ وتكمل دون توقف.

## 🛠️ التطوير المستقبلي

بنية الكود منظمة في دوال مستقلة، مما يسهل إضافة:

- استخراج **جميع** المنتجات تلقائياً (مرر رقماً كبيراً جداً، أو أضف وضع `--all`).
- فلترة حسب السعر أو نسبة الخصم (أضف شرطاً داخل `collect_products`).
- تصدير إلى قاعدة بيانات **SQLite** (أضف دالة `save_sqlite` بجانب دوال الحفظ الحالية).
- إرسال النتائج إلى **Telegram / Discord / API** (أضف دالة إرسال تستقبل قائمة المنتجات النهائية).
